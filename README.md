# Reinforcement_learning
Reinforcement Learning – RL là phương pháp trong khoa học dữ liệu và trí tuệ nhân tạo cho phép một (agent học cách đưa ra quyết định bằng cách tương tác với (environment. Thông qua quá trình thử và sai, tác nhân nhận phần thưởng (reward) hoặc phạt, từ đó dần học được chiến lược tối ưu (policy) nhằm tối đa hóa tổng phần thưởng theo thời gian.
